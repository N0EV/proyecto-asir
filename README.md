# Proyecto Final de Ciclo - ASIR

> [!WARNING]
> Este repositorio, su temática y su estructura se encuentran en fase inicial y estrictamente provisional. El contenido se reestructurará por completo una vez se defina el anteproyecto.

## Descripción
Este repositorio está destinado al desarrollo del proyecto final de estudios para el **C.F.G.S. en Administración de Sistemas Informáticos en Red (ASIR)** en Castilla y León. El espacio albergará la documentación técnica, configuraciones e infraestructura del proyecto.

## Estructura del Repositorio (Provisional)
*   `/deploy`: Scripts de instalación, aprovisionamiento o automatización de la infraestructura.
*   `/config`: Archivos de configuración de los servicios, servidores y políticas de red.
*   `/docs`: Documentación técnica, esquemas/diagramas de red y memoria del proyecto.
