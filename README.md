# Proyecto ASIR

>[!WARNING]
> Este repositorio podrá sufrir modificaciones a lo largo del proceso debido a que se trata de una estructura inicial para asentar las bases del proyecto, a la espera de recibir más información y la asignación del tutor.

Repositorio para el proyecto intermodular de ASIR en el IES Venancio Blanco, adaptado a la nueva ley educativa de FP aplicada en Castilla y León. El marco normativo general se rige por el [Real Decreto 659/2023](https://boe.es) y, de forma específica para la ordenación de los títulos de Grado Superior, por el [Real Decreto 500/2024](https://www.boe.es/diario_boe/txt.php?id=BOE-A-2024-10682).


## Ramas
Este repositorio constará de dos ramas, `main` y `dev`, para lograr un flujo de trabajo más limpio.

- **`main`**: Para el trabajo ya probado y que se ha verificado que funciona.
- **`dev`**: Para el trabajo que está en pruebas y construyéndose.
